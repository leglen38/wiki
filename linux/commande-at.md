# La commande AT
Cette commande permet de plannifier n'importer quelle tache très rapidement dans être obligé de passer par crontab.
Quelles commandes utiles :

1. Redémarrer le serveur aujourd'hui à 13h45

```bash
at 13:45
reboot
```
